üéÆ Database Control Examples

You can now modify the UI entirely through SQL:

Add New Section

INSERT INTO configuration_ui (collection, ui_type, sort) 
VALUES ('new_collection', 'button_grid', 12);

Reorder Sections

UPDATE configuration_ui SET sort = 1 WHERE collection = 'frame_colors';
UPDATE configuration_ui SET sort = 4 WHERE collection = 'hanging_techniques';

Change Render Style

UPDATE configuration_ui SET ui_type = 'dropdown' WHERE collection = 'drivers';

Remove Section

DELETE FROM configuration_ui WHERE collection = 'hanging_techniques';
